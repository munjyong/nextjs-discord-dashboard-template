# A simple Discord dashboard to manage servers. Built with NextJS & Discord.js.

## Features:

### NextJS:

-   Server-side rendering & static site generation (Hybrid)
-   Dynamic routing
-   API routes
-   Layouts
-   Image optimization
-   CSS modules

### Discord.js:

-   Authentication with OAuth2
-   Fetch all guilds
-   Guild member management (kick, timeout, ban)
